EDA Dashboard (Streamlit)

Interactive dashboard to explore and visualize CSV datasets.

Features
	•	Data overview and missing value analysis
	•	Downloadable summary statistics
	•	Correlation analysis (Pearson, Spearman, Kendall)
	•	Visualizations: histogram, box plot, scatter plot, pie chart, line chart

Tech Stack
Python, Streamlit, Pandas, NumPy, Matplotlib, Seaborn
